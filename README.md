# Quantum Noise Analysis and Error Mitigation using Qiskit

## Overview

This project investigates the impact of quantum noise on quantum circuits and evaluates error mitigation techniques using Qiskit.

## Objectives

- Study common quantum noise channels
- Analyze fidelity degradation
- Implement error mitigation methods
- Compare ideal, noisy, and mitigated results

## Tools

- Python
- Qiskit
- Qiskit Aer
- NumPy
- Matplotlib
